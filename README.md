# Smart-Resume-Analyzer-with-ATS-Score
An interactive web-driven Streamlit application built to analyze resumes, evaluate matching criteria for specific job descriptions, and approximate an ATS compatibility score using NLP parsing layers.
## 🚀 Features
- **PDF Processing**: Clean, quick text parsing using `pdfplumber`.
- **ATS Evaluation Engine**: Dynamic word scoring models analyzing structural components.
- **Entity & Skill Detection**: Extracts key technical phrases using tokenized parsing (`spaCy`).
- **Downloadable Diagnostics**: Generates text summaries outlining necessary structural fixes.

## 🛠️ Technology Stack
- **Frontend**: Streamlit
- **Text Processing**: pdfplumber
- **Natural Language Processing**: spaCy, NLTK
- **Data Engineering**: Pandas[resume_analyzer.py](https://github.com/user-attachments/files/29587996/resume_analyzer.py)
